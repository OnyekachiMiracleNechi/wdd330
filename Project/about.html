What is Poise Enterprise?

Poise Enterprise is a smart eCommerce shopping assistant built to help users compare clothing items across various retailers. It saves you time by centralizing product search, ratings, pricing, and reviews from different sources.

APIs Used

  • IHerb Product Data API
  • Flipkart (Optional)

Developer Info

Created by Onyekachi Miracle Nechi for the WDD 330 Final Project. This app is designed using modular JavaScript, real-world APIs, and responsive web design.
